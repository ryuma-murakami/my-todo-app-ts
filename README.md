# My TODO App

A simple TODO app built with React + Vite + TypeScript + Tailwind CSS.

## Features

- Add, edit, delete tasks
- Move tasks to trash before permanent deletion
- Restore tasks from trash
- Empty the trash
- Mark tasks as completed
- Responsive design with Tailwind CSS

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Run ESLint and Prettier checks
npm run lint
npm run format
```
